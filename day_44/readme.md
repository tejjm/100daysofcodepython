CSS Intermediate Project – Box Model and Fonts

A practical project designed to reinforce learning about CSS colors, font properties, and the CSS box model, including margin, padding, and border. The app uses a clean dark theme and elegant fonts to showcase responsive styling and layout techniques.

Features
Applied text color and background colors for contrast

Used Google Fonts integration for custom typography

Manipulated CSS box model properties: margins, padding, and borders

Controlled image styling with borders and spacing

Achieved horizontal centering with margin properties

How It Works
Included Google Fonts link in HTML head for font loading

Styled body with background and text colors

Defined font family, weight, and style using CSS for specific elements

Managed layout using width and margin to center container

Styled images using borders and margins for visual separation

Getting Started
Save index.html and style.css in the same folder

Place image assets in assets/images/ folder as referenced

Open index.html in a browser to see styled page

Modify CSS properties to customize and experiment

What I Practiced & Learned
Leveraging Google Fonts for beautiful typography

Applying and understanding CSS box model—margin, padding, and border

Styling images with borders and whitespace for cleaner layouts

Using percentage-based widths and margins for responsive design
